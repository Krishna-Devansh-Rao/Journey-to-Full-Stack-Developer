- Paragraph -> <p> </p>

--------------------------------------------------------------------------------------------------------
2) PARAGRAPH TAG 
- <p> </p>
- The <p> tag defines a paragraph.
- It is a block-level element, so each paragraph starts on a new line.
- Browsers automatically add margin (space) above and below paragraphs.
- The opening tag is <p> and the closing tag is </p>.
--------------------------------------------------------------------------------------------------------
3) Heading Elements
- Heading elements in HTML are used to define titles and headings on a webpage. HTML provides six levels of headings, from <h1> to <h6>.
<h1> → Largest and most important heading
<h2> → Second largest heading
<h3> → Third largest heading
<h4> → Fourth largest heading
<h5> → Fifth largest heading
<h6> → Smallest and least important heading

HTML (LEVEL 1) PART - B
--------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------
1) List elements 
List elements in HTML are used to display a collection of related items. HTML provides three types of lists:
- Ordered List (<ol>) – Displays items in a numbered sequence.
- Unordered List (<ul>) – Displays items with bullet points.
- Description List (<dl>) – Displays terms and their descriptions.

Key Points
<ol> creates a numbered list.
<ul> creates a bulleted list.
<li> defines a list item (used inside <ol> and <ul>).
<dl> creates a description list.
<dt> defines the term.
<dd> defines the description of the term.
--------------------------------------------------------------------------------------------------------
2) Attributes in HTML
Attributes provide additional information about HTML elements. They are written inside the opening tag and usually consist of a name and a value.

| Attribute | Description                                   | Example                                  |
| --------- | --------------------------------------------- | ---------------------------------------- |
| `href`    | Specifies the URL of a link                   | `<a href="https://example.com">Link</a>` |
| `src`     | Specifies the source of an image, video, etc. | `<img src="photo.jpg">`                  |
| `alt`     | Provides alternative text for an image        | `<img src="cat.jpg" alt="Cat">`          |
| `id`      | Assigns a unique identifier to an element     | `<p id="intro">Hello</p>`                |
| `class`   | Assigns one or more class names               | `<div class="container"></div>`          |
| `style`   | Adds inline CSS styles                        | `<p style="color:red;">Text</p>`         |
| `title`   | Displays extra information as a tooltip       | `<p title="Welcome">Hover me</p>`        |

--------------------------------------------------------------------------------------------------------
3) Anchor Elements
- The Anchor element is used to create hyperlinks that connect one webpage to another webpage, a file, an email address, or a specific section of the same page. The anchor element is represented by the <a> tag.

- <a href="URL">Link Text</a>

Key Points
- The <a> tag is used to create hyperlinks.
- The href attribute specifies the destination URL or file.
- target="_blank" opens the link in a new tab.
- Anchor elements can link to:
- Other websites
- Webpages within the same website
- Email addresses
- Files and downloadable documents
--------------------------------------------------------------------------------------------------------
4) Image Element 
- The Image element is used to display images on a webpage. It is represented by the <img> tag.
- <img src="image.jpg" alt="Description of Image">

Key Points
The <img> tag is used to display images on a webpage.
It is a self-closing (void) element and does not require a closing </img> tag.
The src attribute specifies the image location.
The alt attribute provides alternative text for accessibility and when an image cannot be displayed.
You can control the image size using the width and height attributes.
--------------------------------------------------------------------------------------------------------
5) Comment in HTML
- Comments are used to add notes or explanations in HTML code. They are not displayed in the web browser and are ignored by the browser during page rendering.
<!-- This is an HTML comment -->

1) INLINE VS BLOCK 
- HTML elements are mainly divided into Block-level elements and Inline elements based on how they are displayed on a webpage.

-----------------------------------------------------------------------------------------------------------------------------------
| **Block Elements**                                                     | **Inline Elements**                                    |
| ---------------------------------------------------------------------- | ------------------------------------------------------ |
| Start on a **new line**.                                               | Do **not** start on a new line.                        |
| Occupy the **full width** available.                                   | Occupy only the **required width**.                    |
| Can contain both block and inline elements (depending on the element). | Usually contain only text or other inline elements.    |
| Used for creating the page structure.                                  | Used for formatting or styling content within a line.  |
| Examples: `<div>`, `<p>`, `<h1>`–`<h6>`, `<ul>`, `<ol>`                | Examples: `<span>`, `<a>`, `<img>`, `<strong>`, `<em>` |
-----------------------------------------------------------------------------------------------------------------------------------

-----------------------------------------------------------------------------------------------------------------------------------
2) DIV ELEMENT 
- The <div> (Division) element is a block-level element used to group and organize other HTML elements into sections. It does not have any visual appearance by default but is commonly used with CSS for styling and JavaScript for adding functionality.

Key Points
The <div> tag is known as the Division element.
It is a container used to organize webpage content.
It is one of the most commonly used HTML elements for building layouts.
By itself, <div> has no visual effect until CSS or JavaScript is applied.

-----------------------------------------------------------------------------------------------------------------------------------

3) SPAN ELEMENT

- The <span> element is an inline element used to group or style a small portion of text or other inline elements. It does not start on a new line and has no visual appearance by default. It is commonly used with CSS and JavaScript.

Key Points
The <span> tag is an inline container.
It is used to style or group small portions of text.
Unlike <div>, it does not create a new line.
<span> is commonly used with CSS and JavaScript for text formatting and interactivity.

-----------------------------------------------------------------------------------------------------------------------------------

4) HR TAG

The <hr> (Horizontal Rule) tag is used to create a horizontal line on a webpage. It represents a thematic break or separation between sections of content.

-----------------------------------------------------------------------------------------------------------------------------------

5) SUP AND SUB TAG 
-The <sup> (Superscript) and <sub> (Subscript) tags are used to display text slightly above or below the normal line of text.

-----------------------------------------------------------------------------------------------------------------------------------
6) SEMANTIC MARKUP 
- Semantic Markup is the practice of using HTML elements that clearly describe the meaning and purpose of the content they contain. Instead of using generic tags like <div> and <span> for everything, semantic markup uses meaningful tags such as <header>, <nav>, <main>, and <footer> to make the webpage more understandable.

Key Points
- Semantic Markup means using HTML elements according to their meaning.
- It improves readability, SEO, and accessibility.
- Common semantic elements include <header>, <nav>, <main>, <section>, <article>, <aside>, and <footer>.
- Semantic markup is a best practice in modern HTML development.
-----------------------------------------------------------------------------------------------------------------------------------

7) SEMANTIC TAGS
- Semantic tags are HTML elements that clearly describe the meaning and purpose of the content they contain. They make the webpage easier to understand for developers, search engines, and assistive technologies such as screen readers.

--------------------------------------------------------------------------------------------------
| Tag         | Purpose                                                                          |
| ----------- | -------------------------------------------------------------------------------- |
| `<header>`  | Defines the header section of a webpage or section.                              |
| `<nav>`     | Defines a navigation menu with links.                                            |
| `<main>`    | Represents the main content of the webpage.                                      |
| `<section>` | Groups related content into a section.                                           |
| `<article>` | Represents an independent piece of content, such as a blog post or news article. |
| `<aside>`   | Contains side content, such as advertisements or a sidebar.                      |
| `<footer>`  | Defines the footer section of a webpage or section.                              |
--------------------------------------------------------------------------------------------------

8) HTML ENTITIES 
- HTML Entities are special codes used to display reserved characters, special symbols, and characters that cannot be typed directly in HTML. An HTML entity begins with an ampersand (&) and ends with a semicolon (;).

---------------------------------------------------
| Entity     | Output | Description               |
| ---------- | ------ | ------------------------- |
| `&lt;`     | `<`    | Less than symbol          |
| `&gt;`     | `>`    | Greater than symbol       |
| `&amp;`    | `&`    | Ampersand symbol          |
| `&quot;`   | `"`    | Double quotation mark     |
| `&apos;`   | `'`    | Apostrophe (single quote) |
| `&nbsp;`   | Space  | Non-breaking space        |
| `&copy;`   | ©      | Copyright symbol          |
| `&reg;`    | ®      | Registered trademark      |
| `&trade;`  | ™      | Trademark symbol          |
| `&euro;`   | €      | Euro symbol               |
| `&dollar;` | $      | Dollar symbol             |
---------------------------------------------------

9) EMMETS
- Emmet is a code abbreviation toolkit that helps developers write HTML and CSS code faster. It is built into code editors like Visual Studio Code, allowing you to generate HTML structures by typing short abbreviations and pressing Tab or Enter.

-----------------------------------------------------------------------------
| Emmet Abbreviation              | Generated HTML                          |
| ------------------------------- | --------------------------------------- |
| `!`                             | Creates a basic HTML5 boilerplate       |
| `h1`                            | `<h1></h1>`                             |
| `p`                             | `<p></p>`                               |
| `div`                           | `<div></div>`                           |
| `ul>li*3`                       | Creates a `<ul>` with 3 `<li>` elements |
| `div.container`                 | `<div class="container"></div>`         |
| `div#header`                    | `<div id="header"></div>`               |
| `a[href="https://example.com"]` | `<a href="https://example.com"></a>`    |

1) TABLE

- An HTML Table is used to display data in rows and columns. It is created using the <table> tag. A table consists of rows (<tr>), headers (<th>), and data cells (<td>).
Key Points
<table> defines a table.
<tr> defines a table row.
<th> defines a table heading (bold and centered by default).
<td> defines table data.
Tables are used to organize tabular data.

-----------------------------------------------------------------------------------------------------------------------------------

2) SEMANTIC TAGS IN TABLE
- HTML provides semantic table elements to organize table content. These elements improve readability, accessibility, and SEO.
Tag	Purpose
<caption>	Adds a title to the table.
<thead>	Groups the table header.
<tbody>	Groups the main table data.
<tfoot>	Groups the table footer.
Key Points
<caption> gives the table a title.
<thead> contains heading rows.
<tbody> contains the main data.
<tfoot> contains summary or footer information.
These tags make tables more structured and meaningful.

-----------------------------------------------------------------------------------------------------------------------------------
3) COLSPAN AND ROWSPAN
- colspan and rowspan are table attributes used to merge cells.

colspan
- The colspan attribute allows a cell to span across multiple columns.

rowspan
- The rowspan attribute allows a cell to span across multiple rows.

Key Points
- colspan merges columns.
- rowspan merges rows.
- They help create complex table layouts.
- Used inside <td> or <th> elements.
-----------------------------------------------------------------------------------------------------------------------------------

4) FORMS
- An HTML Form is used to collect user input such as text, email, password, phone number, and more. Forms are created using the <form> tag and contain different form elements.
Key Points
<form> is used to create a form.
Forms collect user information.
Data can be sent to a server using the action and method attributes.
Forms contain various input elements.
Common Form Elements
Element	Purpose
<form>	Creates a form.
<input>	Creates different input fields.
<label>	Defines a label for an input field.
<textarea>	Creates a multi-line text box.
<select>	Creates a drop-down list.
<option>	Defines options inside a drop-down.
<button>	Creates a clickable button.
<fieldset>	Groups related form elements.
<legend>	Adds a title to a fieldset.
<datalist>	Provides predefined suggestions for an input.
<output>	Displays the result of a calculation or action.
